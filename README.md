## Objective

The objective of this task was to perform passive reconnaissance on a publicly accessible website using only publicly available information. The assessment included WHOIS lookup, DNS enumeration, website technology identification, HTTP security header analysis, and robots.txt and sitemap analysis.

## Note

This activity was conducted for educational purposes only. No unauthorized access, scanning, or exploitation was performed.
